# 2026 New Year Wish Jar

An interactive New Year Wish Jar web project built with HTML, CSS and JavaScript.

## Live Demo
https://chipper-biscotti-448853.netlify.app

## About the Project
This project is a simple and interactive web experience where users can open a virtual envelope and write their wishes for the new year.  
It focuses on clean UI, smooth animations, and responsive design.
It has been optimized to work smoothly on both desktop and mobile devices.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Features
- Interactive envelope animation
- Smooth transitions and visual effects
- Fully responsive design
- Deployed on Netlify

## Deployment
The project is deployed using Netlify and automatically updated via GitHub.
